# Employee Management Frontend (Mini Project 2)

This folder contains the API-connected frontend for Mini Project 2.

## How to run

1. Start the API from `EmployeeManagement.Api` with `dotnet run`
2. Open `frontend/index.html` using Live Server
3. Login with:
   - `admin / admin123`
   - `viewer / viewer123`

## Notes

- Uses `http://localhost:5000/api` from `js/config.js`
- Supports role-based UI, dashboard refresh, server-side search/filter/sort, and pagination
- The root project files remain the standalone Mini Project 1 frontend
